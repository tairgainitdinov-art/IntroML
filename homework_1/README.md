Отчёт по лабораторной работе №1
Тема: Анализ данных с использованием библиотеки Pandas
Цель работы:

Освоить базовые приёмы работы с библиотеками Pandas, NumPy, Matplotlib и Seaborn для анализа табличных данных на примере датасета Adult.

Ход работы
1. Загрузка и подготовка данных

В ходе работы был использован датасет Adult, содержащий информацию о доходах, возрасте, образовании, профессии и других характеристиках взрослых людей.
Данные были загружены с помощью функции pd.read_csv():

import pandas as pd
df = pd.read_csv('/content/adult_data.csv')

После загрузки выполнен первичный просмотр структуры таблицы и типов данных.

2. Предварительный анализ

Были проведены базовые операции:

просмотр первых строк датасета (df.head()),

определение размеров таблицы (df.shape),

получение общей информации о столбцах (df.info()),

расчёт основных статистических характеристик (df.describe()).

Данные содержат категориальные и числовые признаки, включая:

возраст (age),

образование (education),

род деятельности (occupation),

уровень дохода (income).

3. Обработка данных

На данном этапе производились:

удаление или обработка пропущенных значений,

преобразование категориальных признаков при необходимости,

фильтрация и группировка данных по определённым условиям (например, анализ распределения доходов по полу или уровню образования).

4. Визуализация

С помощью библиотек Matplotlib и Seaborn были построены графики для визуального анализа зависимостей между признаками.
Примеры визуализаций:

гистограммы распределений возрастов,

диаграммы по доходу и образованию,

тепловые карты корреляций числовых признаков.

import matplotlib.pyplot as plt
import seaborn as sns
plt.rcParams["figure.figsize"] = (18, 10)
sns.countplot(data=df, x='education', hue='income')

5. Анализ полученных результатов

В ходе анализа были получены следующие наблюдения:

уровень дохода часто коррелирует с уровнем образования и стажем работы;

мужчины в среднем имеют более высокий доход, чем женщины;

наибольшая доля людей с высоким доходом наблюдается среди лиц с высшим образованием и в определённых профессиях (например, технических и управленческих).
