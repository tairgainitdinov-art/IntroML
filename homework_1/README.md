### Ход работы

#### 1. Загрузка и подготовка данных
Для выполнения работы был использован датасет Adult, содержащий сведения о возрасте, образовании, профессии, поле и уровне дохода людей.  

Данные были загружены с помощью функции `pd.read_csv()`:

```python
import pandas as pd
df = pd.read_csv('/content/adult_data.csv')
df.head()
```

После загрузки были выполнены базовые операции:
- просмотр первых строк (`df.head()`),
- получение общей информации (`df.info()`),
- вычисление статистических характеристик (`df.describe()`).

---

#### 2. Предварительный анализ
На данном этапе проведено исследование структуры данных:
- определено количество строк и столбцов,
- выявлены категориальные и числовые признаки,
- проверено наличие пропусков и дубликатов.

Основные признаки датасета:
- **age** — возраст,  
- **education** — уровень образования,  
- **occupation** — профессия,  
- **income** — уровень дохода (<=50K или >50K).  

---

#### 3. Обработка данных
Проведена очистка и подготовка данных к анализу:
- удалены или скорректированы пропущенные значения,
- выполнено преобразование категориальных переменных,
- проведена фильтрация и группировка данных для анализа зависимостей между признаками.

---

#### 4. Визуализация данных
Для наглядного представления результатов использовались библиотеки **Matplotlib** и **Seaborn**.  

Примеры визуализаций:
- распределение людей по уровню образования и доходу;
- сравнение возрастных групп;
- тепловая карта корреляций признаков.

```python
import matplotlib.pyplot as plt
import seaborn as sns

plt.rcParams["figure.figsize"] = (18, 10)
sns.countplot(data=df, x='education', hue='income')
plt.title('Распределение уровня дохода по образованию')
plt.show()
```

---

#### 5. Анализ результатов
По результатам анализа установлено:
- более высокий уровень дохода чаще наблюдается у людей с высшим образованием;
- мужчины имеют более высокий средний доход, чем женщины;
- наибольшая доля людей с доходом >50K встречается среди специалистов технических и управленческих профессий.
